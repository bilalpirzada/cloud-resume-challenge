# Cloud Resume Challenge

This repository contains my implementation of the Cloud Resume Challenge, where I am learning AWS and DevOps by building and deploying a real-world cloud application.

## Current Progress

* ✅ Created an AWS account and IAM user
* ✅ Created an Amazon S3 bucket
* ✅ Built a static resume website using HTML, CSS, and JavaScript
* ✅ Enabled static website hosting on Amazon S3
* ✅ Configured bucket policies for public access
* ✅ Deployed the website successfully

## What I Learned

Through this stage of the project, I learned:

* Why static websites should be hosted on Amazon S3 instead of a virtual machine.
* How Amazon S3 static website hosting works.
* Why S3 buckets are private by default.
* The difference between Block Public Access and Bucket Policies.
* Basic Git and GitHub workflow for version control.
* Configure GitHub Actions with OIDC deployment
